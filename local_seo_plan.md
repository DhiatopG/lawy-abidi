# Plan d'Optimisation SEO Local - Maitre Yosra Abidi

Ce plan détaille les étapes nécessaires pour améliorer la visibilité de votre site web dans les recherches locales (Ariana, Tunis, Tunisie).

## 1. Optimisation du Contenu (On-Page)
- [x] **Balise H1** : S'assurer que le titre principal de la page d'accueil contient le lieu.
  - *Cible* : "Avocat à Ariana" (ou "Lawyer Ariana")
- [x] **Mots-clés Locaux** : Intégrer "Ariana", "Tunis", et "Tunisie" naturellement dans les paragraphes de services.
- [x] **Services Spécifiques** : Créer des sections détaillées pour chaque domaine (ex: Droit Immobilier, Droit Civil) avec des références au contexte juridique tunisien.

## 2. Métadonnées & Balises HTML
- [x] **Title Tag** : Optimiser pour inclure le nom, la profession et la ville.
  - *Exemple* : `Best Lawyer Ariana | Avocat Immobilier à Ariana | Near Me`
- [x] **Meta Description** : Rédiger une description incitative mentionnant la localisation et un appel à l'action.
- [x] **Open Graph** : Configurer les balises pour Facebook/LinkedIn avec une image professionnelle et un titre localisé.

## 3. Signaux Techniques Locaux
- [x] **Schéma JSON-LD (Complété)** :
  - [x] Type: `Attorney`
  - [x] Nom: `Maitre Yosra Abidi`
  - [x] Adresse: `Carnoy Center, B23 Rue Tarek Ibn Zied, Ariana 2091`
  - [ ] *Optionnel* : Ajouter les réseaux sociaux (`sameAs`) et les coordonnées GPS (`geo`).
- [x] **NAP Consistency** : Vérifier que le Nom, l'Adresse et le Téléphone dans le footer sont strictement identiques à ceux de la fiche Google Business Profile.
- [x] **Lien "Click-to-Call"** : Transformer le numéro de téléphone en lien cliquable pour mobile.

## 4. Expérience Utilisateur & Trust
- [x] **Google Maps** : Intégrer une carte Google Maps interactive dans la section "Contact" ou le footer.
- [x] **Témoignages** : Mettre en avant des avis clients mentionnant des cas résolus en Tunisie pour renforcer la crédibilité locale.
- [x] **Vitesse & Mobile** : Optimiser le temps de chargement (essentiel pour le SEO local sur mobile).
